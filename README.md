# Japidei Tap

## How do I install these formulae?

`brew install japidei/tap/<formula>`

Or `brew tap japidei/tap` and then `brew install <formula>`.

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
